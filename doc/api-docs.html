Stock Price
Realtime API

REST API for fetching real-time stock prices. Supports two data sources — Yahoo Finance (no key) and Finnhub (real-time with API key).

v1 Yahoo Finance

No API Key Required

Public Yahoo Finance endpoint. Up to 15-min delay on some markets.

/api/v1/stock/:symbol
v2 Finnhub

Real-time Data

Requires FINNHUB_API_KEY. True real-time quotes + company profile.

/api/v2/stock/:symbol

⚙ Tech Stack

Layer Technology
Runtime Node.js (ESM)
Framework Express.js 4
HTTP Client native fetch (Node 18+)
Config dotenv
Data Source v1 Yahoo Finance public API
Data Source v2 Finnhub REST API

▶ Quick Start

1. Install dependencies 

cd project
npm install

2. Create .env file 

FINNHUB_API_KEY=your_finnhub_api_key_here
PORT=3000

3. Run the server 

# production
npm start

# auto-reload (development)
npm run dev

API v1

Yahoo Finance No API Key

May have up to 15-minute delay on some markets. Real-time for US markets during trading hours.

GET /api/v1/stock/:symbol

Fetch a single stock by ticker symbol.

Example Request 

curl http://localhost:3000/api/v1/stock/AAPL

Response 200 

{
  "success": true,
  "data": {
    "symbol": "AAPL",
    "name": "Apple Inc.",
    "price": 189.30,
    "previousClose": 188.05,
    "change": 1.25,
    "changePercent": "0.66%",
    "open": 188.50,
    "high": 190.10,
    "low": 187.90,
    "volume": 52340000,
    "marketCap": 2940000000000,
    "currency": "USD",
    "exchange": "NasdaqGS",
    "marketState": "REGULAR",
    "fiftyTwoWeekHigh": 199.62,
    "fiftyTwoWeekLow": 164.08,
    "timestamp": "2026-05-10T10:30:00.000Z"
  }
}
GET /api/v1/stocks?symbols=SYM1,SYM2,...

Fetch multiple stocks in a single request. Max 20 symbols.

Query param: symbols — comma-separated ticker list

Example Request 

curl "http://localhost:3000/api/v1/stocks?symbols=AAPL,TSLA,MSFT,GOOGL"

Response 200 

{
  "success": true,
  "meta": {
    "requested": 4,
    "found": 4,
    "failed": 0,
    "source": "Yahoo Finance",
    "timestamp": "2026-05-10T10:30:00.000Z"
  },
  "results": [
    { "status": "success", "symbol": "AAPL", "price": 189.30, "...": "..." },
    { "status": "error",   "symbol": "FAKE", "error": "Symbol \"FAKE\" not found on Yahoo Finance" }
  ]
}

! Error Handling

All errors return a consistent JSON body with success: false.

Error Response Format 

{
  "success": false,
  "error": "error message here"
}
Status Cause
400 Missing symbol / invalid format / more than 20 symbols provided
404 Symbol not found on the data source
503 Data source (Yahoo Finance / Finnhub) unresponsive
500 Internal server error

Error Examples 

{ "success": false, "error": "Symbol \"FAKE\" not found on Yahoo Finance" }
{ "success": false, "error": "Symbol parameter is required" }
{ "success": false, "error": "Too many symbols. Maximum is 20, you provided 25" }
{ "success": false, "error": "Finnhub rate limit exceeded — try again shortly" }
{ "success": false, "error": "FINNHUB_API_KEY environment variable is not set" }

Partial Errors in Multi-Stock

If some symbols fail, the request still succeeds. Each result is reported individually. 

{
  "success": true,
  "meta": { "requested": 3, "found": 2, "failed": 1 },
  "results": [
    { "status": "success", "symbol": "AAPL", "price": 189.30 },
    { "status": "success", "symbol": "TSLA", "price": 175.00 },
    { "status": "error",   "symbol": "FAKE", "error": "Symbol \"FAKE\" not found on Finnhub" }
  ]
}
